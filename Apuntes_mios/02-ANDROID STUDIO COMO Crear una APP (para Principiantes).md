# ANDROID STUDIO: COMO Crear una APP (para Principiantes) 📲 [Tutorial]

> Es el [vídeo](https://youtu.be/BQaxPwZWboA?list=PLNdFk2_brsRdDzgiJSamO4vC0cAiRXcH4) sobre instalación de Android Studio y crear una App básica.

*   Explica dónde encontrar los recursos necesarios para desarrolladores de Android, incluyendo el sitio web oficial de desarrolladores de Android que ofrece herramientas, tutoriales y otros recursos. 

*   Presenta Android Studio como la única herramienta necesaria para desarrollar aplicaciones Android, la cual se puede descargar del sitio de desarrolladores.

*   Explica que Android Studio está basado en JetBrains, usando predominantemente Kotlin como lenguaje de programación para desarrollo de apps Android. 

*   Explica el proceso para crear un nuevo proyecto en Android Studio utilizando las plantillas integradas.

*   Se enfoca en crear un nuevo proyecto de Android Studio seleccionando el nombre de la aplicación, el nombre del paquete y el lenguaje de programación, así como la versión mínima de Android a utilizar.

*   Recomienda usar Kotlin porque es el lenguaje del futuro y seleccionar Android 5 como versión mínima porque cubre el 85% de dispositivos.

*   Proporciona una visión general del entorno de desarrollo de Android Studio y sus diversas opciones, como abrir y guardar archivos, compilar el proyecto y ejecutar la aplicación.

*   Cubre cómo crear un emulador para ejecutar la aplicación y navegar la jerarquía del proyecto. 

*   Muestra cómo detener la aplicación y menciona brevemente las diferentes carpetas que componen el proyecto de Android Studio.

*   Discute los diversos archivos y opciones en Android Studio que son cruciales para desarrollar una app Android. 

*   El archivo principal es el manifiesto Android, donde se configuran las diferentes pantallas y el nombre de la aplicación. 

*   Menciona que cada archivo de actividad en Android tiene un archivo de diseño correspondiente que determina el diseño de la pantalla.

*   Otros archivos importantes incluyen scripts de gradle y archivos de recursos.

*   Explora las opciones disponibles en el editor y la consola, incluyendo la capacidad de depurar código e inspeccionar valores.

*   Recomienda aprender Kotlin como el lenguaje de programación principal para desarrollar aplicaciones Android.

*   Invita a los espectadores a visitar su canal de YouTube y explorar tutoriales de programación más avanzados.

*   Concluye el video agradeciendo a los espectadores y diciendo adiós.
